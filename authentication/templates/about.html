Welcome to Pitch

What is PITCH

PITCH is an online platform that helps musicians find gigs. A venue posts an event that require a musician and musicians can apply to play at that event.

How to use Pitch effectively

    Update Profile

  • Once registered, Please go to your profile and click on update profile.

    • Select User type

  • if you are a Venue, please selece Venue from Type user drop down menu on update profile page.
    • This will allow you to submit events on our event board as a Venue and select musicians to play at this event.
  • if you are a Musician, please selece Musician from Type user drop down menu
    • This will Allow you to see the events that require a musician and can apply to play at those events. You can also post an Ad if you are available and would like to play at any event.
